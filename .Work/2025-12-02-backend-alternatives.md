## 1. Firebase Firestore ⭐⭐⭐⭐⭐

### 장점
- ✅ **Google 인프라**: 안정적이고 빠름 (한국 리전 지원)
- ✅ **실시간 리스너**: Supabase와 유사한 실시간 기능
- ✅ **무료 티어**: 일일 읽기 50,000회, 쓰기 20,000회
- ✅ **인증 내장**: 필요시 쉽게 추가 가능
- ✅ **SDK 완성도**: React 통합 우수
- ✅ **보안 규칙**: Firestore Security Rules로 접근 제어

### 단점
- ⚠️ **비용 구조**: 사용량 초과 시 비용 발생 가능
- ⚠️ **학습 곡선**: Security Rules 이해 필요

### 구현 난이도
- **쉬움**: Supabase와 유사한 구조
- Hook만 교체하면 됨

### 비용
- **무료**: 일일 읽기 50K, 쓰기 20K
- **유료**: $25/월부터 (Blaze 플랜)

### 추천 시나리오
- Supabase 대신 가장 쉬운 대체재
- Google 서비스 선호 시

---

## 2. PocketBase ⭐⭐⭐⭐

### 장점
- ✅ **완전 무료**: 자체 호스팅 시 무제한
- ✅ **실시간 기능**: WebSocket 지원
- ✅ **가벼움**: 단일 실행 파일
- ✅ **관리자 UI**: 내장 대시보드
- ✅ **인증 내장**: 사용자 관리 쉬움
- ✅ **SQLite 기반**: 간단한 배포

### 단점
- ⚠️ **자체 호스팅 필요**: 서버 관리 필요
- ⚠️ **한국 리전 없음**: 직접 서버 구축 필요
- ⚠️ **운영 부담**: 서버 관리, 백업 등

### 구현 난이도
- **보통**: 서버 설정 필요
- Vercel/Netlify Functions로 래핑 가능

### 비용
- **무료**: 자체 호스팅 시
- **호스팅 비용**: Railway ($5/월), Render (무료 티어)

### 추천 시나리오
- 서버 관리 가능
- 완전한 제어 원함
- 장기 운영 계획

---

## 3. Airtable ⭐⭐⭐

### 장점
- ✅ **스프레드시트 UI**: 비개발자도 관리 가능
- ✅ **간단한 설정**: API 키만 있으면 됨
- ✅ **무료 티어**: 월 1,200 레코드

### 단점
- ❌ **실시간 없음**: 폴링 필요 (5초마다 체크)
- ⚠️ **제한적 무료**: 레코드 수 제한
- ⚠️ **비용**: 초과 시 $20/월

### 구현 난이도
- **쉬움**: REST API만 사용
- 실시간 대신 폴링 구현 필요

### 비용
- **무료**: 월 1,200 레코드
- **유료**: $20/월 (Plus 플랜)

### 추천 시나리오
- 실시간 불필요
- 스프레드시트로 관리 선호
- 메시지 수가 적음

## 4. Google Sheets API ⭐⭐

### 장점
- ✅ **완전 무료**: Google 계정만 있으면 됨
- ✅ **익숙한 UI**: 스프레드시트로 관리
- ✅ **설정 간단**: API 키 발급만

### 단점
- ❌ **실시간 없음**: 폴링 필요
- ⚠️ **속도 제한**: 분당 100회 요청
- ⚠️ **보안**: API 키 노출 위험
- ⚠️ **CORS 이슈**: 프록시 서버 필요할 수 있음

### 구현 난이도
- **보통**: CORS 처리 필요
- Vercel/Netlify Functions로 래핑 권장

### 비용
- **무료**: 완전 무료

### 추천 시나리오
- 완전 무료 필요
- 실시간 불필요
- 간단한 프로젝트

## 5. JSON 파일 + GitHub Pages ⭐⭐

### 장점
- ✅ **완전 무료**: GitHub 무료
- ✅ **간단함**: 파일만 수정
- ✅ **버전 관리**: Git 히스토리

### 단점
- ❌ **실시간 없음**: 새로고침 필요
- ❌ **쓰기 불가**: GitHub Pages는 읽기 전용
- ⚠️ **GitHub Actions 필요**: 자동화 필요
- ⚠️ **공개 저장소**: 메시지가 공개됨

### 구현 난이도
- **어려움**: GitHub Actions 설정 필요
- 쓰기 기능 구현 복잡

### 비용
- **무료**: 완전 무료

### 추천 시나리오
- 정적 메시지만 표시
- 실시간 불필요
- 완전 무료 필요
