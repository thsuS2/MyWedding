# 작업 기록 - 2026-01-28

## 1) AccountSection 토글 UI 개선 (세그먼트 슬라이더 + 탭 언더라인)

### 요구사항
- 토글 배경은 **은은하게**
- 언더라인은 **active에만** 표시
- 레이아웃/정보 구조는 유지하고 **토글 UI만** 개선

### 구현 내용
- `src/components/account/AccountSection.jsx`
  - 토글 컨테이너에 상태 클래스(`is-groom`/`is-bride`) 부여
  - 접근성 개선: `role="tablist"`, 버튼 `role="tab"`, `aria-selected` 적용
- `src/components/account/AccountSection.css`
  - 토글 컨테이너에 **은은한 슬라이더(선택 배경)**를 `::before`로 구현 (좌/우 이동)
  - active 버튼에만 **언더라인**을 `::after`로 표시
  - `:focus-visible` 스타일 추가

### 수정 파일
- `src/components/account/AccountSection.jsx`
- `src/components/account/AccountSection.css`
