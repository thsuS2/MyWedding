# 버튼 사용 현황 확인

## 현재 상태

### 공통 버튼 스타일 (globals.css)
- `.btn-primary`: 핑크 배경, 흰색 텍스트, 그림자
- `.btn-secondary`: 흰색 배경, 핑크 테두리, 호버 시 핑크 배경

### 개별 버튼 스타일 (각 컴포넌트 CSS)

#### 1. MainSection
- **`.btn-contact`**: 핑크 배경, 큰 버튼 (1rem 2rem)
- **사용**: 연락하기 버튼
- **스타일**: `.btn-primary`와 유사하지만 개별 정의

#### 2. MessageBoard
- **`.btn-write-message`**: 핑크 배경, 큰 버튼 (1rem 2rem)
- **`.btn-primary`**: 폼 제출 버튼 (globals.css 사용)
- **사용**: 메시지 작성하기 버튼, 폼 제출 버튼
- **스타일**: `.btn-contact`와 거의 동일

#### 3. Footer
- **`.btn-kakao`**: 핑크 배경, 큰 버튼 (1rem 2rem)
- **사용**: 카카오톡 공유하기 버튼
- **스타일**: `.btn-contact`와 거의 동일

#### 4. AccountSection
- **`.btn-copy`**: 핑크 배경, 작은 버튼 (0.75rem 1.5rem)
- **사용**: 계좌번호 복사하기 버튼
- **스타일**: 작은 크기, 복사됨 상태 스타일 추가

#### 5. MapSection
- **`.btn-primary`**: globals.css 사용 (카카오맵 버튼)
- **`.btn-secondary`**: globals.css 사용 (네이버지도 버튼)
- **`.btn-rsvp`**: 핑크 배경, 큰 버튼 (15px 30px)
- **사용**: 지도 버튼들, 참석의사 전달하기 버튼
- **스타일**: `.btn-rsvp`는 `.btn-contact`와 유사

#### 6. RSVPModal
- **`.rsvp-modal-close`**: 닫기 버튼 (X)
- **`.rsvp-check-button`**: 체크 버튼
- **사용**: 모달 닫기, 폼 체크 버튼

#### 7. GallerySection
- **`.slide-nav`**: 이전/다음 버튼
- **`.modal-close`**: 모달 닫기 버튼
- **사용**: 갤러리 네비게이션, 모달 닫기

#### 8. BottomSheet
- **`.bottom-sheet-close`**: 닫기 버튼 (X)
- **사용**: 하단 패널 닫기

#### 9. IntroSection
- **`.intro-mute-btn`**: 음소거 버튼
- **사용**: 동영상 음소거 토글

## 분석 결과

### 공통점
- 대부분의 주요 버튼들이 핑크 배경(`var(--point-pink)`) 사용
- 호버 시 `transform: translateY(-2px)` 효과 공통
- `border-radius: 8px` 공통
- 그림자 효과 사용

### 차이점
- **크기**: 큰 버튼(1rem 2rem) vs 작은 버튼(0.75rem 1.5rem)
- **그림자**: 일부는 `var(--shadow-md)`, 일부는 `var(--shadow-sm)`
- **호버 효과**: 일부는 배경색 변경, 일부는 transform만

### 권장사항
1. **큰 버튼들 통일**: `.btn-contact`, `.btn-write-message`, `.btn-kakao`, `.btn-rsvp`는 모두 `.btn-primary`로 통일 가능
2. **작은 버튼**: `.btn-copy`는 크기가 달라서 별도 유지 또는 `.btn-primary`에 size variant 추가
3. **닫기 버튼**: `.rsvp-modal-close`, `.bottom-sheet-close`, `.modal-close`는 공통 스타일로 통일 가능

## 결론
현재 모든 버튼이 인라인 `<button>` 태그를 사용하고 있으며, Button 컴포넌트는 사용하지 않음 (이전에 삭제됨).
대부분의 버튼 스타일이 유사하지만 각 컴포넌트에 개별 정의되어 있어, 공통 스타일로 통일하면 유지보수가 쉬워질 수 있음.
