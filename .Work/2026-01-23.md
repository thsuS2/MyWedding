## 1. Vercel Kakao API 환경변수 설정

### 작업 내용
- Vercel 대시보드에서 `VITE_KAKAO_APP_KEY` 환경변수 등록
- Kakao JavaScript Key를 사용하여 카카오톡 공유 기능 활성화
- `VITE_KAKAO_MAP_API_KEY` 환경변수도 동일한 JavaScript Key로 등록

### 확인 사항
- Kakao JavaScript Key는 클라이언트 사이드에서 공개되어도 안전함
- Vercel 경고 메시지는 정상이며 무시해도 됨
