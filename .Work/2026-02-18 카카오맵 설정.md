# 카카오맵 API 키 설정 가이드

## 현재 상태
- `.env` 파일에 `VITE_KAKAO_MAP_API_KEY`가 이미 설정되어 있습니다.
- 로컬 개발 환경에서는 이 값이 자동으로 사용됩니다.

## 로컬 개발 환경 설정

### 1. .env 파일 확인
프로젝트 루트에 `.env` 파일이 있고, 다음 내용이 포함되어 있어야 합니다:

```env
VITE_KAKAO_MAP_API_KEY=0e3116db4709c31c21ffa5114ea0c151
```

### 2. 개발 서버 재시작
환경 변수를 변경한 경우, 개발 서버를 재시작해야 합니다:

```bash
# 서버 중지 (Ctrl + C)
# 서버 재시작
npm run dev
```

## Vercel 배포 환경 설정

Vercel에 배포할 때는 환경 변수를 별도로 설정해야 합니다.

### 1. Vercel 대시보드에서 설정
1. Vercel 대시보드 접속: https://vercel.com
2. 프로젝트 선택
3. **Settings** → **Environment Variables** 메뉴로 이동
4. 다음 환경 변수 추가:
   - **Name**: `VITE_KAKAO_MAP_API_KEY`
   - **Value**: `0e3116db4709c31c21ffa5114ea0c151`
   - **Environment**: Production, Preview, Development 모두 선택
5. **Save** 클릭
6. **Deployments** 탭에서 최신 배포를 **Redeploy** (환경 변수 변경 후 재배포 필요)

### 2. Vercel CLI로 설정 (선택사항)
```bash
# Vercel CLI 설치 (없는 경우)
npm i -g vercel

# 환경 변수 설정
vercel env add VITE_KAKAO_MAP_API_KEY
# 값 입력: 0e3116db4709c31c21ffa5114ea0c151
# 환경 선택: Production, Preview, Development

# 재배포
vercel --prod
```

## 카카오맵 API 키 발급 방법 (새로 발급이 필요한 경우)

1. **카카오 개발자 콘솔 접속**: https://developers.kakao.com
2. **내 애플리케이션** → 애플리케이션 선택 (또는 새로 만들기)
3. **앱 키** 탭에서 **JavaScript 키** 확인
4. **플랫폼** 탭에서 **Web 플랫폼 등록**
   - 사이트 도메인: `http://localhost:5173` (개발), `https://jisooyusinwedding.vercel.app` (프로덕션)
5. **앱 키** → **JavaScript 키**를 복사하여 `.env` 파일의 `VITE_KAKAO_MAP_API_KEY` 값으로 설정

## 문제 해결

### 지도가 보이지 않는 경우

1. **브라우저 콘솔 확인**
   - 개발자 도구(F12) → Console 탭에서 에러 메시지 확인
   - `⚠️ 카카오맵 API 키가 설정되지 않았습니다` 메시지가 보이면 환경 변수 미설정

2. **환경 변수 확인**
   ```javascript
   // 브라우저 콘솔에서 확인
   console.log(import.meta.env.VITE_KAKAO_MAP_API_KEY);
   ```
   - `undefined`가 나오면 환경 변수가 제대로 로드되지 않은 것

3. **개발 서버 재시작**
   - `.env` 파일 변경 후에는 반드시 개발 서버를 재시작해야 함

4. **API 키 유효성 확인**
   - 카카오 개발자 콘솔에서 JavaScript 키가 활성화되어 있는지 확인
   - 플랫폼 도메인이 올바르게 등록되어 있는지 확인

5. **CORS 에러 확인**
   - 브라우저 콘솔에서 CORS 관련 에러가 있는지 확인
   - 카카오 개발자 콘솔의 플랫폼 설정에서 도메인이 등록되어 있어야 함

## 참고사항

- `.env` 파일은 `.gitignore`에 포함되어 있어 Git에 커밋되지 않습니다.
- Vercel 배포 시에는 반드시 Vercel 대시보드에서 환경 변수를 설정해야 합니다.
- 환경 변수 이름은 `VITE_`로 시작해야 Vite에서 접근 가능합니다.
