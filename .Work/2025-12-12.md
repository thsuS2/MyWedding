## 파비콘 이모티콘 적용

### 요구사항
- 기존 아이콘 파일 대신 이모티콘을 파비콘으로 사용
- 💍 (반지) 이모티콘 선택

### 구현 내용

#### 1. index.html 수정
- 기존: `<link rel="icon" type="image/svg+xml" href="/vite.svg" />`
- 변경: Data URI 방식으로 💍 이모티콘을 SVG로 감싸서 적용

```html
<link rel="icon" href="data:image/svg+xml,<svg xmlns=%22http://www.w3.org/2000/svg%22 viewBox=%220 0 100 100%22><text y=%22.9em%22 font-size=%2290%22>💍</text></svg>" />
```

### 구현 방식
- **Data URI 기반 SVG**: 이모티콘을 SVG로 감싸서 Data URI로 직접 삽입
- 별도 파일 생성 불필요
- 브라우저 호환성 우수

### 장점
- 즉시 적용 가능
- 별도 파일 관리 불필요
- 대부분의 모던 브라우저에서 동작
- 작은 크기에서도 명확하게 보임

### 테스트 방법
1. 개발 서버 실행 후 브라우저 탭에서 파비콘 확인
2. 다양한 브라우저에서 테스트 (Chrome, Safari, Firefox 등)
3. 모바일 브라우저에서도 확인

### 완료 상태
✅ 완료

## 혼주에게 연락하기 기능 추가

### 요구사항
- 혼주에게 전화/문자 연락 기능 추가
- 모바일 웹에서 바로 전화/문자 걸 수 있도록 구현
- 신랑/신부에게 직접 연락하는 기능 (parent-name-wrapper 내부)
- 혼주에게 연락하기 모달 (신랑측/신부측 혼주 각각 엄마/아빠)
- 전화/문자 버튼은 아이콘으로 표시

### 구현 내용

#### 1. 데이터 구조 추가 (`wedding.js`)
- `COUPLE` 객체에 전화번호 필드 추가:
  - `groom.phone`: 신랑 전화번호
  - `bride.phone`: 신부 전화번호
  - `groom.parents.fatherPhone`: 신랑 아빠 전화번호
  - `groom.parents.motherPhone`: 신랑 엄마 전화번호
  - `bride.parents.fatherPhone`: 신부 아빠 전화번호
  - `bride.parents.motherPhone`: 신부 엄마 전화번호

#### 2. MainSection.jsx 수정
- `useState`로 모달 상태 관리 추가
- `formatPhoneForLink` 헬퍼 함수: 전화번호에서 하이픈 제거하여 `tel:`/`sms:` 링크 형식으로 변환
- `parent-name-wrapper` 내부에 신랑/신부에게 연락하는 아이콘 버튼 추가
  - `PiPhone` 아이콘: 전화 링크 (`tel:`)
  - `PiChatCircle` 아이콘: 문자 링크 (`sms:`)
- "혼주에게 연락하기" 버튼 추가
- 혼주 연락 모달 컴포넌트 구현:
  - 신랑측 혼주 섹션 (엄마, 아빠 각각 전화/문자 버튼)
  - 신부측 혼주 섹션 (엄마, 아빠 각각 전화/문자 버튼)
  - 모달 오버레이 클릭 시 닫기
  - 닫기 버튼 (✕)

#### 3. MainSection.css 스타일 추가
- `.contact-link`: 신랑/신부 연락 버튼 스타일 (원형 아이콘 버튼)
- `.contact-parents-btn`: 혼주에게 연락하기 버튼 스타일
- `.parents-modal-overlay`: 모달 오버레이 스타일 (RSVPModal 참고)
- `.parents-modal-content`: 모달 컨텐츠 스타일
- `.parents-side-section`: 혼주 측별 섹션 스타일
- `.parents-contact-item`: 각 혼주 연락 항목 스타일
- `.parents-contact-btn`: 혼주 연락 버튼 스타일 (원형 아이콘 버튼)
- 모바일 반응형 스타일 추가

### 구현 방식
- **모바일 네이티브 링크**: `tel:` 및 `sms:` 프로토콜 사용으로 모바일에서 바로 전화/문자 앱 실행
- **아이콘 기반 UI**: React Icons의 `PiPhone`, `PiChatCircle` 사용
- **모달 패턴**: 기존 RSVPModal과 유사한 스타일로 일관성 유지
- **접근성**: `aria-label` 속성으로 스크린 리더 지원

### 주요 기능
1. **신랑/신부 직접 연락**: parent-name-wrapper 내부의 아이콘 버튼으로 바로 연락
2. **혼주 연락 모달**: "혼주에게 연락하기" 버튼 클릭 시 모달 표시
3. **모바일 최적화**: 모바일에서 버튼 클릭 시 전화/문자 앱 자동 실행
4. **반응형 디자인**: 모바일/데스크톱 모두 최적화된 UI
